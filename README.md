##Este projeto se trata do desafio RPG da DIO. 

A fim de explorar a utilziação de classes, foram utilizados cards de jogo do Pokémon. 
Cada pokémon possui HP, duas habilidades e cada habilidade causa uma certa quantidade de dano. 
Além disso, cada pokémon possui um tipo. Os tipos pode aumentar ou diminuir a quantidade de dano que cada habilidade poderá causar. 

No caso do exemplo, há um pokémon do tipo elétrico e outro do tipo água. 
O pokémon do tipo elétrico causa o dobro de dano naquele de tipo água. 

Dessa forma, foi criada uma classe abstrata para os pokemons. 
Depois foi criada uma classe para o pokémon elétrico e outra para o de água. 

Foi criada uma função específica para quando um pokémon elétrico ataca um de água. 

Foi criada uma classe para pokémon de água para que futuramente o projeto possa ser escalado com outros pokémons/tipos.
